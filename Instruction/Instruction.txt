PROBLEM STATEMENT

E-commerce platforms and retailers often face challenges creating engaging, high-quality product descriptions for consumer packaged goods. Manual content creation is time-consuming, costly, and prone to inconsistency, which can negatively impact sales and brand image. There is a growing need for an Al-driven solution that automates generation of persuasive, accurate product descriptions aligned with product features and brand tone. Current tools lack flexibility and often fail to capture essential attributes or marketing appeal. Automating this process would accelerate content production, ensure consistency, and enhance customer engagement across product catalogs.

Data Considerations (Guidelines):

Structured product datasets containing attributes such as size, ingredients, usage instructions, and branding in CSV or JSON formats; existing product descriptions for training prompt examples; public product catalogs and marketing copy for reference; synthetic data generation to expand attribute variations; preprocessing to normalize attribute formats and filter irrelevant data; dataset volume sufficient to represent diverse product types; no personal data involved, minimizing compliance concerns.

Feel free to generate synthetic data where appropriate using the provided environment

SOLUTION EXPECTATIONS

A functional Al prototype that generates persuasive and accurate product descriptions from structured input data via a simple UI; features include batch processing, customization of tone, and output formatting; deliverables include working code, user guide, and demo video demonstrating description quality and batch capabilities; KPIs focus on description relevance and user satisfaction; demo to highlight ease of use and quality of outputs; basic API integration to support product data updates.

if upload_file: 
with tempfile.NamedTemporaryFile(delete=False, suffix=".pdf") as temp_file: 
temp_file.write(upload_file.read()) 
temp_file_path = temp_file.name 
# Step 1: Extract text 
raw_text = extract_text(temp_file_path) 
# Step 2: Chunking  
   text_splitter = RecursiveCharacterTextSplitter(chunk_size=1000, chunk_overlap
=200) 
   chunks = text_splitter.split_text(raw_text) 
   # Step 3: Embed and store in Chroma 
   with st.spinner("Indexing document..."): 
       vectordb = Chroma.from_texts(chunks, embedding_model,persist_directory="
./chroma_index") 
       vectordb.persist() 
   # Step 4: RAG QA Chain 
   #retriever 
= vectordb.as_retriever(search_type="similarity", search_kwargs={"k": 5}) 
   retriever = vectordb.as_retriever() 
   rag_chain = RetrievalQA.from_chain_type( 
       llm=llm, 
       retriever=retriever, 
       return_source_documents=True 
   ) 
   # Step 5: Ask summarization prompt 
   summary_prompt = "Please summarize this document based on the key topics:" 
   with st.spinner("Running RAG summarization..."): 
       # result = rag_chain.run(summary_prompt) 
       result = rag_chain.invoke(summary_prompt) 
   st.subheader("     Summary") 
   st.write(result)
